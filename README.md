# Portfolio – Adrian

Frontendowe portfolio zbudowane w React + Vite. Skupia się na technologiach: Java, JavaScript, SQL (PostgreSQL) oraz MongoDB. Zawiera sekcje: O mnie, Umiejętności, Projekty, Certyfikaty.

## Uruchomienie lokalne

```bash
npm install
npm run dev
```

## Budowanie

```bash
npm run build
npm run preview
```
